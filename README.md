## 📖 项目简介

**微光·求职搭子** 是一款基于 DeepSeek 大模型技术的智能求职辅助工具。它不再只是简单的职位筛选器，而是你的**AI 求职参谋**。通过深度理解你的简历和职位描述（JD），它能帮你：

- ⚡ **瞬间识破**职位描述背后的"潜台词"
- 🎯 **精准匹配**核心优势，提供面试重点展示建议
- 🛡️ **智能识别**外包、培训贷、保险销售等潜在风险
- 📝 **生成面试押题**，预测面试官可能问的问题

告别海投，精准狙击。让每一份简历投递都掷地有声。

## ✨ 核心功能

### 1. ⚡ 深度职位剖析
在 Boss直聘 职位详情页，AI 助手为你生成详尽的战术报告：

- **匹配度评分**: 0-100 分直观展示你与职位的契合度
- **核心优势**: 提炼你简历中与该职位最匹配的亮点（面试时重点展示）
- **风险识别**: 识别外包、驻场、培训贷、保险销售等潜在风险
- **面试押题**: 根据 JD 预测面试官可能会问的 3-5 个犀利问题，并提供回答思路
- **薪资揭秘**: 通过数据提取技术，获取隐藏的薪资范围信息

### 2. 🚀 列表智能巡航
在职位列表页开启"巡航模式"，自动扫描当前页面所有职位：

- **视觉分级**: 高匹配度职位显示**绿色边框**，中等匹配显示**蓝色边框**
- **自动过滤**: 低匹配度或活跃度低的职位显示**灰色半透明**
- **智能暂停**: 发现高匹配职位后自动暂停扫描（可配置阈值）
- **批量分析**: 支持一键分析整个列表页的所有职位

### 3. 🔬 实验室模式（Lab Mode）
高级数据提取功能，提供更深入的职位分析：

- **网络请求拦截**: Hook 技术，获取更多隐藏数据
- **深度薪资探测**: 提取未公开的薪资信息
- ⚠️ **风险提示**: 可能存在账号封禁风险，需谨慎使用

### 4. 🛡️ 隐私与安全

- **数据脱敏**: 简历发送给 AI 前，自动抹除姓名、手机号、身份证等敏感信息
- **本地存储**: 你的简历数据仅保存在浏览器本地，不上传第三方服务器
- **透明处理**: 所有数据处理过程均在浏览器端完成，可追踪

## 🏗️ 技术架构

### 技术栈

- **Chrome Extension Manifest V3**: 现代扩展架构
- **DeepSeek API**: 大语言模型服务
- **JavaScript (ES6+)**: 主要开发语言
- **Chrome Storage API**: 本地数据持久化
- **html2canvas**: 截图功能支持

### 核心组件

```
boss-agent/
├── manifest.json           # 扩展配置清单 (Manifest V3)
├── background.js           # Service Worker，处理 API 调用和核心逻辑
├── content.js              # 内容脚本，注入 Boss直聘 页面 (6,106 行)
├── injected_probe.js       # 注入脚本，深度数据提取 (Hook 技术)
├── spa_monitor.js          # SPA 导航监控
├── popup.html/popup.js     # 扩展弹窗界面和设置管理
├── style.css               # UI 样式文件
├── html2canvas.min.js      # 第三方截图库
└── CLAUDE.md               # 项目开发文档
```

### 数据流架构

```
┌─────────────────┐
│   popup.js      │ ← 设置管理、能量领取、AI 配置生成
└────────┬────────┘
         │ chrome.storage / chrome.tabs.sendMessage
         ├─────────────────────────────────────┐
         ↓                                     ↓
┌─────────────────┐                   ┌─────────────────┐
│ background.js   │ ←─ API 调用 ────→ │ DeepSeek API    │
│ Service Worker  │                   │   (AI 分析)     │
└────────┬────────┘                   └─────────────────┘
         │ chrome.runtime.sendMessage
         ↓
┌─────────────────┐
│   content.js    │ ← DOM 监听、UI 注入、任务协调
│  (主控制器)     │
└────────┬────────┘
         │ window 事件通信
         ↓
┌─────────────────┐
│injected_probe.js│ ← 网络请求拦截、隐藏数据提取
│  + spa_monitor  │
└─────────────────┘
```

### 关键技术实现

#### 1. SPA 导航处理
- 监听 URL 变化和 DOM 更新
- 自动识别页面切换（列表页 ↔ 详情页）
- 动态注入/移除 UI 元素

#### 2. 数据提取策略
- **被动扫描**: DOM 解析 + 全局变量检查
- **主动探测**（实验室模式）: 网络请求拦截 + Hook 技术
- **隐藏薪资**: 提取页面加密数据

#### 3. 消息传递机制
- **popup → background**: 设置读写、能量领取
- **popup → content**: 直接触发职位分析
- **content → background**: API 调用请求
- **content → injected**: 数据提取请求
- **injected → content**: 自定义事件回调
