## 📖 简介

**微光·求职搭子** 是一款带夜视功能的职场翻译机，基于 DeepSeek 大模型技术。它不再只是简单的筛选工具，而是你的**求职参谋**。通过深度理解你的简历和职位描述（JD），它能帮你瞬间识破职位背后的“潜台词”，精准匹配核心优势，并提供“毒辣”的面试押题。

## ✨ 核心功能

### 1. ⚡ 深度职位剖析
在职位详情页，AI 助手会为你生成一份详尽的战术报告：
- **匹配度评分**: 0-100 分直观展示契合度。
- **核心优势**: 提炼你简历中与该职位最匹配的亮点（面试时重点展示）。
- **避坑指南**: 识别外包、驻场、长期出差等潜在风险。
- **面试押题**: 根据 JD 预测面试官可能会问的 3 个犀利问题，并提供回答思路。

### 2. 🚀 列表智能巡航
在职位列表页开启“巡航模式”，插件会自动扫描当前页面的所有职位：
- **高亮推荐**: 高匹配度职位显示**绿色**边框。
- **自动过滤**: 低匹配度或活跃度低的职位显示灰色半透明，节省注意力。

### 3. 🛡️ 隐私安全
- **数据脱敏**: 简历发送给 AI 前，自动抹除姓名、手机号等敏感隐私。
- **本地存储**: 你的简历数据仅保存在你的浏览器本地，不会上传至任何第三方服务器。

## 🛠️ 安装与使用指南

### 第一步：安装插件
1. 下载本插件的压缩包并解压（或者 `git clone` 源码）。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`。
3. 打开右上角的 **“开发者模式”** 开关。
4. 点击左上角的 **“加载已解压的扩展程序”**。
5. 选择解压后的文件夹（包含 `manifest.json` 的目录）。

### 第二步：初始化配置
1. 点击浏览器右上角的插件图标（建议固定在工具栏）。
2. 在弹出的面板中：
   - **输入简历**: 粘贴你的简历纯文本内容。
   - **配置算力**: 选择使用自有 Key 或微光免费能量。
3. 点击 **“✨ 一键生成我的专属配置”**。
4. 等待约 120 秒，AI 会根据你的简历生成专属的分析逻辑。

### 第三步：开始求职
1. 打开 Boss直聘 网站。
2. **职位列表页**: 点击右侧悬浮球的“🚀”按钮，开启列表巡航。
3. **职位详情页**: 点击右侧悬浮球的“⚡”按钮，查看深度分析报告。

**Q: 插件收费吗？**
A: **完全免费**。本插件旨在帮助求职者打破信息差，所有功能均免费开放。
